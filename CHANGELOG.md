# Changelog

## 0.0.2

* Adds `AxisX` and `AxisY` components
* Exports `getLayout` and `getSettings` so that consumers can implement their own SvelteVis components

## 0.0.1

* Initial release
* Adds basic structural components for responsive visualisations:
    * `Visualisation`
    * `Header`
    * `Footer`
    * `Graphic`
    * `Svg`
    * `Plot`
