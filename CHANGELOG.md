# Changelog

## 0.0.16

Version 0.0.16 represents a new consolidated baseline version of the package. Earlier versions of the package were developed against pre-release versions of Svelte 5 and did not support TypeScript. This is the first working version that supports both Svelte 5 and TypeScript. Earlier versions of the package have been deprecated and removed from NPM.

* The full set of components exported in this release are:
    * `ColorSchemeObserver`
    * `Visualisation`
    * `Header`
    * `Footer`
    * `Graphic`
    * `Svg`
    * `Gridlines`
    * `AxisX`
    * `AxisY`
    * `Plot`
    * `CircleGeometry`
    * `LineGeometry`
    * `Popup`
